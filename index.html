Catalogo degli investimenti e delle riforme previste dal Piano Nazionale Ripresa e Resilienza (PNRR) 2021 con apposite risorse finanziarie

Consulta le informazioni sulle risorse del PNRR. La fonte delle informazioni è il documento in formato PDF del Governo italiano pubblicato online ad aprile 2021.
Visualizzazione delle informazioni su questa piattaforma a cura di Ciro Spataro (lic. )
Grazie al repository GitHub di Riccardo Saporiti (lic. ) in cui ha creato il file in formato CSV degli interventi finanziati dal PNRR.
Nota → nell'aprire i dati del documento PDF del Governo, si è tenuto in considerazione gli "investimenti" e le "riforme" che prevedono uno stanziamento economico. Le riforme per le quali non è prevista una spesa nel PNRR non compaiono in questo catalogo.

Strumenti

Gli strumenti principali usati per realizzare questa visualizzazione sono quattro:

  • Simile Exhibit, per realizzare l'interfaccia di navigazione del catalogo;
  • Twitter Boostrap, per dare un po' di stile alle pagine pubblicate;
  • GDrive Sheet, per raccogliere e distribuire i dati che fanno da sorgente;
  • GitHub Pages, per l'hosting dei file che costituiscono questa visualizzazione.
